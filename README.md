## 🕹 Gameplay

Experience a lightweight, western-themed runner inspired by classics like *Subway Surfers*, reimagined for a more zen experience.

* **🤠 The Hero:** Take control of a rugged cowboy runner.
* **🛤 The Path:** Navigate through beautifully designed western tracks.
* **🪙 The Loot:** Collect as many coins as possible during your run.
* **⚠️ The Danger:** Test your reflexes by avoiding obstacles and environmental hazards.
* **🏁 The Goal:** Reach the finish line safely to progress.

## 📦 Installation

### Android
1.  Navigate to the **Releases** section of this repository.
2.  Download the latest `.apk` file.
3.  Enable **"Install from Unknown Sources"** in your device settings if prompted.
4.  Open the file and tap **Install**.
5.  Launch **Dustway Run** and start playing! 🎮
